Fichiers STL STL files

⚠️ Saisie manuelle requise pour les STL

⚠️ Manual entry required for STL files

Les fichiers STL contiennent uniquement la géométrie — aucune donnée de tranchage. Vous devez saisir manuellement les informations d'impression.

STL files contain only geometry — no slicing data. You must manually enter the print information.

⚠️

Pas de calcul automatique avec les STL

No automatic calculation with STL files

Contrairement aux fichiers 3MF tranchés, un fichier STL ne contient pas le temps d'impression, le poids ou le remplissage. Ces données dépendent de votre trancheur (PrusaSlicer, Bambu Studio, Cura…) et doivent être renseignées à la main après tranchage.

Unlike sliced 3MF files, an STL file does not contain print time, weight, or infill. These values depend on your slicer (PrusaSlicer, Bambu Studio, Cura…) and must be entered manually after slicing.

🔢

Données à saisir manuellement

Data to enter manually

  • → Temps d'impression (en heures/minutes)
  • → Poids de matière utilisée (en grammes)
  • → Taux de remplissage (en %)
  • → Print time (hours/minutes)
  • → Material weight used (in grams)
  • → Infill percentage (%)

Conseil pratique

Practical tip

Tranchez votre STL dans votre logiciel habituel (Cura, PrusaSlicer…), relevez le temps estimé, le poids et le remplissage affichés, puis renseignez ces valeurs dans le calculateur.

Slice your STL in your usual software (Cura, PrusaSlicer…), note the estimated time, weight and infill displayed, then enter these values in the calculator.

Fichiers 3MF tranchés Sliced 3MF files

✅ Import automatique des données de tranchage

✅ Automatic import of slicing data

Un fichier 3MF tranché contient toutes les informations réelles de votre machine. Importez-le pour un calcul précis et automatique.

A sliced 3MF file contains all the real data from your machine. Import it for accurate, automatic calculation.

💡

Important : utiliser le fichier tranché du plateau

Important: use the sliced plate file

Dans Bambu Studio, ce fichier s'appelle "Exporter fichier tranché du plateau". Il contient le temps réel, la consommation matière et les paramètres machine. Chaque imprimante et chaque configuration de plateau donnera des données différentes.

In Bambu Studio, this file is called "Export sliced file of the plate". It contains the real print time, material usage and machine parameters. Each printer and plate configuration will produce different data.

Différence par machine

Per-machine difference

Le temps d'impression, le coût et le remplissage varient d'une machine à l'autre. Pour un résultat précis, associez toujours l'imprimante correcte à chaque calcul ou utilisez un fichier 3MF tranché par cette machine.

Print time, cost and infill vary between machines. For accurate results, always associate the correct printer with each calculation, or use a 3MF file sliced by that machine.
